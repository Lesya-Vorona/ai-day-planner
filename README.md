# AI-планер дня

Mobile-first todo-застосунок: диктуй або пиши все, що в голові, а далі це перетворюється на структуровані задачі.

Це поки що **каркас**: 3 екрани, нижня таб-навігація, стан у `localStorage`. Без бекенду і без AI-парсингу — розбір задач додамо пізніше.

## Екрани

- **Захопити** (`/`) — велике поле вводу + кнопка диктування (Web Speech API, де підтримується браузером)
- **Вхідні** (`/inbox`) — список захоплених задач, звідси можна перенести задачу на сьогодні
- **Сьогодні** (`/today`) — чекліст задач на сьогодні

## Розробка

```bash
npm install
npm run dev
```

Відкрий [http://localhost:3000](http://localhost:3000). Для мобільного вигляду відкрий інструменти розробника й увімкни емуляцію мобільного пристрою.

## Стек

Next.js (App Router) · React · TypeScript · Tailwind CSS 4 · state у React + `localStorage`
